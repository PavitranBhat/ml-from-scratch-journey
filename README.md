# 🚀 ML From Scratch Journey

This repository documents my structured journey of learning **Machine Learning from scratch**, focusing on both **theoretical understanding** and **practical implementation** of core algorithms.

---

## 🎯 Objective

To build a strong foundation in Machine Learning by:

* Understanding concepts deeply (not just using libraries)
* Implementing algorithms step-by-step
* Practicing real-world problem solving
* Maintaining a consistent learning log

---

## 📚 Topics Covered

### 🔹 Day 1: Basics

* Introduction to Machine Learning
* Types of ML (Supervised, Unsupervised)
* Basic workflow

### 🔹 Day 2: Linear Regression

* Concept of regression
* Cost function (MSE)
* Gradient descent intuition
* Implementation

### 🔹 Day 3: Multiple Linear Regression

* Multiple features
* Model interpretation
* Real-world use cases

### 🔹 Day 4: Logistic Regression

* Classification vs regression
* Sigmoid function
* Log loss
* Decision boundary
* Confusion matrix

### 🔹 Day 5: K-Nearest Neighbors (KNN)

* Distance-based learning
* Euclidean distance
* Choosing K
* Feature scaling importance
* Implementation with evaluation
